# Project Title

Sample Wordcount Java app to count words in a file using Map and Reduce functions in hadoop

## Getting Started

Clone the project

### Prerequisites

Java 8
Maven

### Installing


Install dependencies

```
mvn clean install compile
```

Build packaged jar

```
mvn package
```
